Number Guessing

The Number Guessing Game is a fun and interactive game in which the computer randomly selects a number within a predefined range, and the player tries to guess it in the fewest attempts possible.

» At the start of the game, the program generates a random number (for example, between 1 and 100).
» The player then enters their guess.
» After each guess, the program provides a hint:
» If the guess is too high, it tells the player to try a smaller number.
» If the guess is too low, it tells the player to try a larger number.
» The process continues until the player correctly guesses the number or it use limited attempts.
» The game can also keep track of the number of attempts taken to guess correctly, encouraging the player to improve.
